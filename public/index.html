Ciudad 3D THREE.js

Ciudad en 3D hecha con Three.js que incluye construcciones, vehículos en movimiento, partículas ambientales y efectos de iluminación. Permite interacción con el mouse y adapta su vista al tamaño de pantalla.

Reproductor de Música

Reproductor de música personalizado desarrollado en JavaScript. Permite al usuario reproducir, pausar y navegar entre distintas canciones, mostrando dinámicamente el título, artista, carátula y el progreso de reproducción. Incluye una barra de progreso interactiva y cambia automáticamente al siguiente tema al finalizar la pista actual.

HTML5 JavaScript CSS

Tetris GameBoy

Un videojuego interactivo inspirado en el Tetris de la GameBoy, desarrollado en JavaScript utilizando la librería p5.js. El usuario puede mover las piezas con las teclas de dirección, hacer que caigan más rápido o rotarlas para encajarlas en el tablero. Lo que se espera de una clasica experiencia completa del juego Tetris.

HTML5 JavaScript CSS P5.js
